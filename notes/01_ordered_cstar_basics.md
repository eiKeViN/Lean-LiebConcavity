# Ordered C⋆-algebras: Structure, Positivity, and CFC

## Typeclass context

The minimal setup used throughout this project:

```lean
variable {A : Type*} [CStarAlgebra A] [PartialOrder A] [StarOrderedRing A]
```

`CStarAlgebra A` bundles `NormedRing`, `StarRing`, `NormedAlgebra ℂ A`,
`ContinuousFunctionalCalculus ℝ A IsSelfAdjoint`, and `T2Space`. The two extra
typeclasses add the ordered structure (`PartialOrder`) and the cone axiom
(`StarOrderedRing`: the positive cone is generated by elements of the form
`star b * b`).

## 1. Positive elements `{a | 0 ≤ a}`

### 1.1 Positive elements are self-adjoint

```
Mathlib/Algebra/Order/Star/Basic.lean  (line 184)
lemma IsSelfAdjoint.of_nonneg {x : R} (hx : 0 ≤ x) : IsSelfAdjoint x
alias LE.le.isSelfAdjoint := IsSelfAdjoint.of_nonneg   -- dot-notation: hx.isSelfAdjoint
```

**Consequence:** whenever `0 ≤ a` is available, `IsSelfAdjoint a` is free —
never carry both hypotheses simultaneously.

### 1.2 The positive cone is convex (but NOT a subspace)

**Convex:** via `convex_Ici` applied at `r = 0`:

```
Mathlib/Analysis/Convex/Basic.lean  (line 251)
theorem convex_Ici (r : β) : Convex 𝕜 (Ici r)
-- typeclasses: [AddCommMonoid β] [PartialOrder β] [IsOrderedAddMonoid β] [Module 𝕜 β] [PosSMulMono 𝕜 β]
```

```lean
example [CStarAlgebra A] [PartialOrder A] [StarOrderedRing A] [PosSMulMono ℝ A] :
    Convex ℝ {a : A | 0 ≤ a} := convex_Ici 0
```

**Not a subspace:** not closed under negation (`0 ≤ a → -(1:ℝ) • a = -a ≤ 0`).
The correct algebraic label is **convex cone**. Mathlib: `ProperCone.convex`
(`Mathlib/Analysis/Convex/Cone/Basic.lean:92`).

### 1.3 Spectrum of a positive element lies in `[0, ∞)`

The instance:
```
Mathlib/Analysis/CStarAlgebra/ContinuousFunctionalCalculus/Basic.lean  (line 351)
instance CStarAlgebra.instNonnegSpectrumClass : NonnegSpectrumClass ℝ A
-- (requires [CStarAlgebra A] [PartialOrder A] [StarOrderedRing A])
```

gives:
```
Mathlib/Algebra/Algebra/Spectrum/Quasispectrum.lean  (line 418)
lemma spectrum_nonneg_of_nonneg (ha : 0 ≤ a) (hx : x ∈ spectrum ℝ a) : 0 ≤ x
```

So `0 ≤ a → spectrum ℝ a ⊆ Set.Ici 0`.

## 2. Self-adjoint elements `{a | IsSelfAdjoint a}`

### 2.1 Form a real linear subspace

```
Mathlib/Algebra/Star/Module.lean  (line 121)
def selfAdjoint.submodule [Semiring R] [StarMul R] [TrivialStar R]
    [AddCommGroup A] [Module R A] [StarAddMonoid A] [StarModule R A] :
    Submodule R A
-- carrier = {a | IsSelfAdjoint a}
```

For `R = ℝ`: `TrivialStar ℝ` holds (ℝ-star is the identity), and `CStarAlgebra A`
provides `StarModule ℝ A`. So `selfAdjoint.submodule ℝ A : Submodule ℝ A` is available.

Closure lemmas in `Mathlib/Algebra/Star/SelfAdjoint.lean`:
- `IsSelfAdjoint.add` — `ha.add hb`
- `IsSelfAdjoint.neg` — `ha.neg`
- `IsSelfAdjoint.smul` — `ha.smul r` for `r : ℝ` (via `StarModule`)

### 2.2 Convex (follows from being a subspace)

No dedicated `convex_isSelfAdjoint` lemma; convexity is automatic from the
submodule structure. Every submodule is convex.

### 2.3 NOT a ℂ-subspace

`i • a` is skew-adjoint when `a` is self-adjoint:
`star (i • a) = conj(i) • star a = -i • a = -(i • a)`.

## 3. CFC and positivity

### 3.1 When is `cfc f a` nonneg?

```
Mathlib/Analysis/CStarAlgebra/ContinuousFunctionalCalculus/Unital.lean  (line 951)
lemma cfc_nonneg {f : R → R} {a : A} (h : ∀ x ∈ spectrum R a, 0 ≤ f x) :
    0 ≤ cfc f a
```

The iff version (requires `NonnegSpectrumClass`):
```
-- line 929
lemma cfc_nonneg_iff [NonnegSpectrumClass R A] (f : R → R) (a : A) ... :
    0 ≤ cfc f a ↔ ∀ x ∈ spectrum R a, 0 ≤ f x
```

**Combined rule:** if `0 ≤ a` (so `spectrum ℝ a ⊆ [0,∞)` by §1.3) and
`f x ≥ 0` for all `x ≥ 0`, then `0 ≤ cfc f a`.

### 3.2 Real powers `a ^ y` for any `y : ℝ`

Definition:
```
Mathlib/Analysis/SpecialFunctions/ContinuousFunctionalCalculus/Rpow/Basic.lean  (line 408)
noncomputable def rpow (a : A) (y : ℝ) : A := cfc (fun x : ℝ≥0 => x ^ y) a
-- notation: a ^ y   (low-priority Pow instance, line 412)
```

Key positivity result (holds for **all** `y : ℝ`, including negative `y`):
```
-- line 419
@[simp] lemma rpow_nonneg {a : A} {y : ℝ} : 0 ≤ a ^ y
```

Because the CFC is taken over `ℝ≥0`, the predicate is `0 ≤ ·` and
`cfc_predicate` gives nonnegativity for free.

**Required import:**
```lean
import Mathlib.Analysis.SpecialFunctions.ContinuousFunctionalCalculus.Rpow.Basic
```

### 3.3 Inverse of a positive element

```
Mathlib/Analysis/CStarAlgebra/ContinuousFunctionalCalculus/Unital.lean  (line 1011)
lemma CFC.inv_nonneg_of_nonneg (a : Aˣ) (ha : 0 ≤ (a : A)) : 0 ≤ (a : A)⁻¹
```

Requires `a` to be a unit (equivalently, `a` is strictly positive / invertible).

The iff:
```
-- line 1014
lemma CFC.inv_nonneg (a : Aˣ) : 0 ≤ (a : A)⁻¹ ↔ 0 ≤ (a : A)
```

### 3.4 Negative rpow does NOT require invertibility

For non-invertible `0 ≤ a`, `a ^ y = 0` when `y < 0` (since `ℝ≥0` rpow sets
`0 ^ y = 0`). The formula `rpow_nonneg` still holds trivially.
For `0 < a` (strictly positive), `a ^ y > 0` and all the usual rpow identities
(`rpow_add`, `rpow_mul`, etc.) apply.

## 5. Relevance to this project

`OperatorConvexOn I f` (defined in `LeanLiebConcavity/Defs.lean`) quantifies over
the domain
```lean
{a : B | IsSelfAdjoint a ∧ spectrum ℝ a ⊆ I}
```
This is the intersection of the ℝ-subspace of self-adjoint elements with a
spectrum condition. When `I` is a convex subset of ℝ, this intersection is convex,
consistent with `ConvexOn` requiring a convex domain.

In `JensenOperator2003`, the hypothesis `0 ≤ aᵢ` can replace `IsSelfAdjoint aᵢ`
(by §1.1 above). The spectrum conditions `spectrum ℝ aᵢ ⊆ I` are then discharged
from `0 ≤ aᵢ` combined with `spectrum_nonneg_of_nonneg` when `I = [0, ∞)`.
